# .env 配置说明

> 逐字段说明每个环境变量的含义、获取方式和示例值。

---

## .env 文件在哪？

- 使用 `setup.sh` 一键脚本：自动创建，按提示填写即可
- 手动部署：从模板复制后编辑

```bash
cp .env.example src/.env
nano src/.env     # 或用任何编辑器打开
```

## 必填项

### DeepSeek API（AI 大脑）

```bash
DEEPSEEK_API_KEY=sk-your-deepseek-api-key
DEEPSEEK_BASE_URL=https://api.lkeap.cloud.tencent.com/v1
DEEPSEEK_MODEL=deepseek-v3
```

| 字段 | 说明 |
|------|------|
| `DEEPSEEK_API_KEY` | API 密钥，从你选择的平台获取 |
| `DEEPSEEK_BASE_URL` | API 地址 |
| `DEEPSEEK_MODEL` | 模型名称 |

**三个平台任选其一**：

| 平台 | BASE_URL | MODEL | 获取方式 |
|------|----------|-------|----------|
| 腾讯云 lkeap（推荐） | `https://api.lkeap.cloud.tencent.com/v1` | `deepseek-v3` | [lkeap 控制台](https://console.cloud.tencent.com/lkeap) → 创建 API Key |
| DeepSeek 官方 | `https://api.deepseek.com/v1` | `deepseek-chat` | [DeepSeek 平台](https://platform.deepseek.com/) → API Keys |
| 其他兼容平台 | 该平台的 OpenAI 兼容地址 | 该平台的模型名 | 见平台文档 |

> 腾讯云 lkeap 推荐理由：国内网络稳定、无需翻墙、开通即用。充值 10 元可用很久。

### 企业微信配置

```bash
WEWORK_CORP_ID=ww1234567890abcdef
WEWORK_AGENT_ID=1000002
WEWORK_CORP_SECRET=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
WEWORK_TOKEN=xxxxxxxxxxxxxxxxxxxx
WEWORK_ENCODING_AES_KEY=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

| 字段 | 获取方式 |
|------|----------|
| `WEWORK_CORP_ID` | 企微后台 → 我的企业 → 企业信息 → 企业ID |
| `WEWORK_AGENT_ID` | 企微后台 → 应用管理 → 你的应用 → AgentId |
| `WEWORK_CORP_SECRET` | 企微后台 → 你的应用 → Secret → 查看（扫码） |
| `WEWORK_TOKEN` | 企微后台 → 你的应用 → 接收消息 → 随机获取 |
| `WEWORK_ENCODING_AES_KEY` | 同上，和 Token 一起生成 |

> 详细图文步骤见 [企微注册与应用创建](企微注册与应用创建.md)

### 用户与管理

```bash
DEFAULT_USER_ID=YourWeWorkUserID
ADMIN_TOKEN=your-admin-secret-token
```

| 字段 | 说明 | 获取方式 |
|------|------|----------|
| `DEFAULT_USER_ID` | 你的企微用户 ID（管理员） | 企微后台 → 通讯录 → 点击你的名字 → 账号 |
| `ADMIN_TOKEN` | 管理后台登录密码 | 自己设一个长密码（如 `my-karvis-admin-2026`） |

## 可选项

### Qwen API（省钱：轻量任务用 Qwen，降低 DeepSeek 调用量）

```bash
QWEN_API_KEY=
QWEN_BASE_URL=https://dashscope.aliyuncs.com/compatible-mode/v1
QWEN_MODEL=qwen-plus-latest
QWEN_VL_MODEL=qwen-vl-max
```

- 不填则所有任务走 DeepSeek
- 注册地址：https://bailian.console.aliyun.com/

### OneDrive（笔记同步到 Obsidian）

```bash
ONEDRIVE_CLIENT_ID=
ONEDRIVE_CLIENT_SECRET=
ONEDRIVE_REFRESH_TOKEN=
```

- 不填 = Lite 模式，笔记保存在项目根目录 `my_life/` 文件夹
- 填写后笔记自动同步到 OneDrive 上的 Obsidian Vault

### 腾讯云 ASR（语音识别）

```bash
TENCENT_APPID=
TENCENT_SECRET_ID=
TENCENT_SECRET_KEY=
```

- 不填则不支持语音转文字
- 在 [腾讯云控制台](https://console.cloud.tencent.com/) → 访问管理 → API 密钥管理 获取

### 天气服务

```bash
SENIVERSE_KEY=
WEATHER_CITY=深圳
```

- 不填则早报不含天气信息
- [心知天气](https://www.seniverse.com/) 注册获取免费 Key

### 其他参数

```bash
DAILY_MESSAGE_LIMIT=50          # 每人每天消息上限（防止成本失控）
WEB_TOKEN_EXPIRE_HOURS=24       # Web 查看链接有效时长（小时）
PROCESS_ENDPOINT_URL=http://127.0.0.1:9000/process  # 保持默认即可
```

```bash

DEEPSEEK_API_KEY=sk-xxxxxxxxx
DEEPSEEK_BASE_URL=https://api.lkeap.cloud.tencent.com/v1
DEEPSEEK_MODEL=deepseek-v3

# 企业微信
WEWORK_CORP_ID=wwxxxxxxxxx
WEWORK_AGENT_ID=1000002
WEWORK_CORP_SECRET=xxxxxxxxx
WEWORK_TOKEN=xxxxxxxxx
WEWORK_ENCODING_AES_KEY=xxxxxxxxx

# 用户
DEFAULT_USER_ID=YourUserID
ADMIN_TOKEN=my-secret-admin-token
```
