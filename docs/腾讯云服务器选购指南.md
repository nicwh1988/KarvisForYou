## 购买后设置

### 1. 找到你的公网 IP

购买完成后，在 [轻量服务器控制台](https://console.cloud.tencent.com/lighthouse/instance) 可以看到实例的 **公网 IP**。

### 2. 开放端口

Karvis 默认用 9000 端口，需要在防火墙中开放：

1. 控制台 → 选择你的实例 → **防火墙**
2. 点击 **添加规则**
3. 填写：
   - 应用类型：自定义
   - 协议：TCP
   - 端口：9000
   - 来源：所有（`0.0.0.0/0`）
4. 确定

### 3. 连接服务器

**方式一：控制台 WebShell（最简单）**
- 实例列表 → 点击 **登录** → 直接在浏览器中操作

**方式二：SSH 连接**
```bash
ssh root@你的公网IP
# 输入密码（购买时设置的，或在控制台重置）
```

**方式三：CodeBuddy 远程连接**
- CodeBuddy 有腾讯云 Lighthouse 插件，可以直接连接操作

### 4. 部署 Karvis

连上服务器后，按照 [部署指南](部署指南.md) 操作。推荐使用 Docker Compose 方式。

## 购买后的完整流程

```
购买服务器 → 开放 9000 端口 → SSH 连接
    → git clone 代码 → 配置 .env → 启动 Karvis
    → 企微后台填 URL（http://公网IP:9000/wework）
    → 配置企业可信 IP
    → 发"你好"验证
```

## 常见问题

### Q: 买完后怎么上传代码到服务器？

直接在服务器上 clone：
```bash
git clone https://github.com/sameencai/KarvisForYou.git
```

### Q: 忘了服务器密码怎么办？

控制台 → 实例 → 重置密码。

### Q: 不想用了能退款吗？

腾讯云轻量服务器支持 5 天内无理由退款（首台）。
