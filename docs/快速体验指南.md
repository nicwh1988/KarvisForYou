# 5 分钟快速体验指南

> 在自己电脑上跑通 Karvis，发出第一条消息。不需要买服务器。

---

## 你需要准备

| 项目 | 说明 |
|------|------|
| 一台电脑 | Mac / Windows / Linux 均可 |
| Python 3.9+ | Mac 自带；Windows 从 [python.org](https://www.python.org/downloads/) 安装（勾选 Add to PATH） |
| Git | Mac 自带；Windows 从 [git-scm.com](https://git-scm.com/) 安装 |
| DeepSeek API Key | [腾讯云 lkeap](https://console.cloud.tencent.com/lkeap) 开通，充值 10 元 |
| 企微应用配置 | 按 [企微注册与应用创建](企微注册与应用创建.md) 拿到 6 个配置值 |

---

## 开始

### 1. 下载代码

```bash
git clone https://github.com/sameencai/KarvisForYou.git
cd KarvisForYou
```

### 2. 一键启动

```bash
chmod +x setup.sh
./setup.sh
```

脚本会自动：
- 检查 Python 版本
- 安装依赖
- 引导你填写 API Key 和企微配置
- 安装 cloudflared
- 启动服务并生成公网 URL

### 3. 看到这个就成功了

```
╔══════════════════════════════════════════════════════════════╗
║  公网 URL 已生成!                                            ║
╚══════════════════════════════════════════════════════════════╝
  你的公网地址: https://xxx-xxx-xxx.trycloudflare.com
  企微后台 URL 填: https://xxx-xxx-xxx.trycloudflare.com/wework
```

### 4. 配置企微回调

1. 打开 [企微管理后台](https://work.weixin.qq.com/wework_admin/frame)
2. 应用管理 → 你的应用 → 接收消息 → 设置 API 接收
3. URL 填：`https://xxx-xxx-xxx.trycloudflare.com/wework`（用上面生成的地址）
4. Token 和 EncodingAESKey 填 `.env` 中的值（setup.sh 引导时已填过）
5. 点击保存

### 5. 发第一条消息

在微信中找到你的企业 → 找到 Karvis → 发一句"你好"。

收到回复 = 成功！

> 如果微信里找不到企业：去企微后台 → 我的企业 → 微信插件 → 用微信扫描"邀请关注"二维码

## Windows 用户

如果 `setup.sh` 无法运行，手动操作：

```bash
cd KarvisForYou/src
pip install -r requirements.txt
copy ..\.env.example .env
# 用记事本打开 .env，填入配置
python app.py
```

cloudflared 另开一个终端：
```bash
# 安装: winget install cloudflare.cloudflared
cloudflared tunnel --url http://localhost:9000
```
