# 企微注册与应用创建

> 从零开始：注册企业微信组织 → 创建自建应用 → 获取所有必要配置信息。全程截图指引。

---

## 第一步：注册企业微信

1. 打开 https://work.weixin.qq.com/ ，点击 **「企业注册」**
2. 选择 **「企业」** 类型（个人也可以注册，不需要营业执照）
3. 填写组织名称（随便起，比如"空空的小团队"）
4. 用微信扫码完成管理员绑定
5. 注册完成，自动进入管理后台

> 注册后你就是这个组织的管理员，拥有所有权限。

---

## 第二步：获取 Corp ID（企业 ID）

1. 登录 [企业微信管理后台](https://work.weixin.qq.com/wework_admin/frame)
2. 左侧菜单 → **我的企业** → **企业信息**
3. 页面底部找到 **企业 ID**，复制保存

```
对应 .env 字段: WEWORK_CORP_ID=ww1234567890abcdef
```

---

## 第三步：创建自建应用

1. 左侧菜单 → **应用管理**
2. 页面下方 **自建** 区域 → 点击 **「创建应用」**
3. 填写信息：
   - **应用 Logo**：上传一张图片（Karvis 的头像）
   - **应用名称**：比如 `Karvis` 或 `AI管家`
   - **可见范围**：选择你自己（之后可以加其他成员）
4. 点击 **「创建应用」**

---

## 第四步：获取 AgentId 和 Secret

创建完成后自动进入应用详情页：

**AgentId**：
- 页面上方直接显示，复制保存

```
对应 .env 字段: WEWORK_AGENT_ID=1000002
```

**Secret**：
- 点击 Secret 旁边的 **「查看」**
- 用企微 App 扫码验证
- 复制显示的 Secret（只展示一次，立即保存）

```
对应 .env 字段: WEWORK_CORP_SECRET=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

> Secret 只展示一次！如果忘了保存，需要点击"重置"生成新的。

---

## 第五步：获取 Token 和 EncodingAESKey

1. 在应用详情页，找到 **接收消息** → 点击 **「设置 API 接收」**
2. 点击 **「随机获取」** 分别生成 Token 和 EncodingAESKey
3. 复制保存这两个值

```
对应 .env 字段:
WEWORK_TOKEN=xxxxxxxxxxxxxxxxxxxx
WEWORK_ENCODING_AES_KEY=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

4. **URL 先空着**，等 Karvis 启动后再回来填

> Token 和 EncodingAESKey 可以反复查看，不像 Secret 那样只展示一次。

---

## 第六步：获取你的 User ID

1. 左侧菜单 → **通讯录**
2. 点击你自己的名字
3. 页面上的 **「账号」** 就是你的 User ID

```
对应 .env 字段: DEFAULT_USER_ID=YourUserID
```

---

## 第七步：在微信中关注你的企业

部署完成后，需要在微信中关注企业才能使用：

1. 管理后台 → **我的企业** → **微信插件**
2. 找到 **「邀请关注」** 区域的二维码
3. 用 **微信** 扫码，确认关注

扫码后回到微信聊天列表，就能看到你的企业名称了。点进去就能找到 Karvis 应用。

---

## 配置汇总

完成以上步骤后，你应该拿到这些值：

| 信息 | 来源 | .env 字段 |
|------|------|-----------|
| 企业 ID | 我的企业 → 企业信息 | `WEWORK_CORP_ID` |
| AgentId | 应用详情页 | `WEWORK_AGENT_ID` |
| Secret | 应用详情页 → 查看 | `WEWORK_CORP_SECRET` |
| Token | 接收消息 → 随机获取 | `WEWORK_TOKEN` |
| EncodingAESKey | 接收消息 → 随机获取 | `WEWORK_ENCODING_AES_KEY` |
| User ID | 通讯录 → 点击自己 | `DEFAULT_USER_ID` |

全部拿到后，进入下一步：[.env 配置说明](.env配置说明.md) 填写配置文件。
