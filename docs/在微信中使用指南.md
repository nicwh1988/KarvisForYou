# 在微信中使用指南

> 部署完成后，如何在微信里找到 Karvis、置顶、日常使用。

---

## 前提

你已经完成了：
- Karvis 服务已启动并运行
- 企微后台回调 URL 配置成功
- 发"你好"能收到回复

如果还没完成，先看 [快速体验指南](快速体验指南.md) 或 [部署指南](部署指南.md)。

---

## 第一步：在微信中关注企业

1. 登录 [企微管理后台](https://work.weixin.qq.com/wework_admin/frame)
2. 左侧菜单 → **我的企业** → **微信插件**
3. 找到 **「邀请关注」** 区域的二维码
4. 用 **微信** 扫码，确认关注

> 这一步把你的微信和企业微信组织关联起来。

---

## 第二步：在微信中找到 Karvis

关注成功后，回到微信聊天列表：

1. 找到你的 **企业名称**（如"空空的小团队"），它显示在聊天列表中
2. 点击进入 → 找到 **Karvis**（或你设置的应用名称）
3. 点击进入聊天页面

> Karvis 在微信中是一个二级入口：微信聊天列表 → 企业 → Karvis。

---

## 第三步：置顶聊天

让 Karvis 始终在企业聊天列表最上方：

1. 进入 Karvis 的聊天页面
2. **长按**聊天标题（或点击右上角设置）
3. 选择 **「置顶聊天」**

置顶后，每次点进企业就能直接看到 Karvis。

---

## 安卓用户额外技巧

安卓微信支持将应用添加到手机桌面：

1. 进入 Karvis 聊天页面
2. 点击右上角 **「...」**
3. 选择 **「添加到桌面」**

这样可以从手机桌面一键直达 Karvis，不用打开微信再找。

> 注意：iOS（iPhone）暂不支持此功能。

---

## 使用入口说明

| 平台 | 入口方式 |
|------|----------|
| 微信（iOS/安卓） | 聊天列表 → 企业名称 → Karvis |
| 安卓微信 | 桌面快捷方式一键直达 |
| 企业微信 App | 工作台 → Karvis |
| Web 管理端 | `http://IP:9000/web/logs`（管理员） |

---

## 邀请其他人使用

想让朋友/家人也用你的 Karvis：

1. 企微管理后台 → **通讯录** → **添加成员**
2. 填写对方信息，邀请加入
3. 对方用微信扫描「邀请关注」二维码
4. 对方在微信中就能找到 Karvis 了

> 每个用户的数据完全独立，互不影响。
